# Análisis Numérico

Repositorio correspondiente a la materia **Análisis Numérico**, cursada durante el segundo Cuatrimestre de 2025.


El repositorio contiene tres entregas desarrolladas durante el cursado,
incluyendo informese e implementaciones en código.

## Contenido

- **Entrega 1**: Capa límite en una ecuación de convección–difusión.
- **Entrega 2**: Ecuación del calor bidimensional mediante diferencias finitas.
- **Entrega 3 (Entrega Final)**: Problema de reacción–difusión 2D resuelto con el método de elementos finitos.

Cada carpeta incluye la consigna correspondiente y el código utilizado.
