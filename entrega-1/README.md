# Entrega 1 — Ejercicio 8 (Guía 1)

Esta entrega corresponde al **Ejercicio 8 de la Guía 1**.

## Descripción del problema

Se estudia la ecuación estacionaria de convección-difusión:

ε U''(x) − U'(x) = f(x),   x ∈ (0,1)

con condiciones de borde de Dirichlet y distintos valores del parámetro ε.

El objetivo es analizar el fenómeno de **capa límite** y el comportamiento
del error numérico para distintos valores de ε y tamaños de malla.

## Contenido

- `entrega_1.jl`: implementación en Julia del método numérico.
- `informe.pdf`: informe con el análisis teórico y resultados (si corresponde).
